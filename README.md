# Welcome to BandStagePlot A Rewrite.

# A tool designed for bands and musicians to create professional-looking stage plots.

### Features
Read "[Features](https://github.com/noahlikesvr/BandStagePlot/wiki/Features)" on the Wiki

### How to Use
1. Visit my website at [thestagedesigner.com](https://thestagedesigner.com/).
2. Click an image to get it on your stage.
3. Drag and drop instruments and equipment into place.
4. Save and share your stage plot with others.

### Support / Contributing
I want to keep everything in github, if you see issues or want other images added submit a PR please!
